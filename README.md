
# Neon Login Authenticator

simple package for authentication using neon project


## URL Reference

#### Login PATH

```http
  GET /neon/login
```
#### Logout PATH

```http
  GET /neon/logout
```

#### Check Session PATH

```http
  GET /neon/check-session
```
#### Check Email PATH

```http
  GET /neon/check-email/{email}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `email`      | `string email` | **Required**. Registered Email |
